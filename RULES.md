# Splendor (宝石の煌き) ルール仕様書

## 1. コンポーネント

### 1.1 トークン (宝石)
6種類存在し、それぞれ以下の個数が用意される。
- **エメラルド (緑)**: 7枚
- **サファイア (青)**: 7枚
- **ルビー (赤)**: 7枚
- **ダイヤモンド (白)**: 7枚
- **オニキス (黒)**: 7枚
- **黄金 (黄/ジョーカー)**: 5枚

※ プレイ人数によって使用する枚数が異なる（後述）。

### 1.2 発展カード
全90枚。レベル1, 2, 3に分かれている。
- **レベル1 (緑)**: 40枚
- **レベル2 (黄)**: 30枚
- **レベル3 (青)**: 20枚

各カードには以下が記載されている：
- **コスト**: 購入に必要な宝石の種類と数。
- **威信ポイント (VP)**: 勝利点（0点の場合もある）。
- **ボーナス**: 購入後にその色の宝石1つ分として永続的にカウントされる宝石の種類。

### 1.3 貴族タイル
全10枚。
- **獲得条件**: 指定された種類のボーナス（発展カード）を所持していること。
- **効果**: 3 威信ポイント (VP)。
- **性質**: アクションを消費せず、条件を満たしたターンの終了時に自動的に獲得。早い者勝ち。

## 2. セットアップ

### 2.1 プレイ人数による調整

| プレイヤー人数 | 宝石トークン (各色) | 黄金トークン | 貴族タイル数 |
| :--- | :--- | :--- | :--- |
| **2人** | 4枚 | 5枚 | 3枚 |
| **3人** | 5枚 | 5枚 | 4枚 |
| **4人** | 7枚 | 5枚 | 5枚 |

### 2.2 盤面配置
1. **発展カード**: 各レベルの山札から4枚ずつ表向きに並べる（計12枚）。
2. **貴族タイル**: 人数に応じた枚数（人数 + 1枚）をランダムに選び、表向きに並べる。
3. **トークン**: 種類ごとに山にして置く。

---

## 3. ゲームの進行

スタートプレイヤーから順に時計回りで手番を行う。
プレイヤーは手番中に以下の**3つのアクションのうち1つだけ**を実行しなければならない。

### 3.1 アクション

#### A. 異なる色の宝石トークンを3枚取る
- 異なる3種類の色の宝石トークンを各1枚ずつ、計3枚取る。
- **条件**: 黄金トークンは取れない。

#### B. 同じ色の宝石トークンを2枚取る
- 同じ色の宝石トークンを2枚取る。
- **条件**: その色のトークンが場に**4枚以上**残っている場合のみ選択可能。

#### C. 発展カードを1枚確保（予約）する
- 場の表向きのカード、または各レベルの山札の一番上のカードを1枚選び、手札に加える。
- 手札の上限は**3枚**。既に3枚持っている場合はこのアクションを選択できない。
- **特典**: 黄金トークンが場にある場合、1枚獲得できる（強制ではないが、通常は取る）。

#### D. 発展カードを1枚購入する
- 場の表向きのカード、または手札（予約済み）のカードを1枚選ぶ。
- カードに記載されたコスト分の宝石トークンを支払う（場に戻す）。
- **ボーナス**: 既に購入済みの発展カードのボーナスは、その色のトークンとしてコストから差し引くことができる。
- **黄金トークン**: 任意の色として代用可能。
- 購入したカードは自分の手元に置き、ボーナスとVPが加算される。
- 場のカードを購入した場合は、即座に対応する山札から補充する。

### 3.2 貴族の訪問
- 手番終了時、貴族タイルの条件（指定されたボーナス数）を満たしているか確認する。
- 満たしている場合、その貴族タイルを1枚獲得する（アクション消費なし）。
- 複数の貴族の条件を同時に満たした場合でも、獲得できるのは**1ターンにつき1枚**まで（プレイヤーが選択）。

### 3.3 トークン所持制限
- ターン終了時、手元のトークン（黄金含む）が**10枚**を超えている場合、10枚になるように選んで場に戻さなければならない。

---

## 4. ゲームの終了

### 4.1 終了条件
- いずれかのプレイヤーの威信ポイント（VP）の合計が**15点以上**になったラウンドで終了フラグが立つ。
- 全プレイヤーが同じ回数の手番を行えるように、スタートプレイヤーの右隣のプレイヤーまで手番を行ってゲーム終了となる。

### 4.2 勝敗判定
- 最もVPが高いプレイヤーの勝利。
- **同点の場合**: 購入した発展カードの枚数が**少ない**プレイヤーの勝利。
- それでも同点の場合は引き分け（またはハウスルール）。
