# Necessary Assets for Splendor App

Although the current implementation uses CSS placeholders, to make the game visually complete, the following image assets are recommended.

## 1. Gem Tokens (Icons)
Format: PNG/SVG (Transparent background)
- `assets/gems/emerald.png`
- `assets/gems/sapphire.png`
- `assets/gems/ruby.png`
- `assets/gems/diamond.png`
- `assets/gems/onyx.png`
- `assets/gems/gold.png`

## 2. Card Assets
Format: JPG/PNG
- **Card Backs**:
  - `assets/cards/back_level_1.jpg` (Green theme)
  - `assets/cards/back_level_2.jpg` (Yellow theme)
  - `assets/cards/back_level_3.jpg` (Blue theme)
- **Gem Icons (Small)**: For card costs (can reuse Token icons or simplified versions).

## 3. Noble Tiles
Format: JPG/PNG (Square or Portrait)
- 10 Unique Noble Portraits:
  - `assets/nobles/noble_1.jpg`
  - ...
  - `assets/nobles/noble_10.jpg`

## 4. UI Elements (Optional)
- `assets/ui/background.jpg` (Game table texture)
- `assets/ui/logo.png` (Game Title)
